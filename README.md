# scratch

Queries Github API to search for users from specified location.

## Prerequisites

You will need [Leiningen][] 2.0.0 or above installed.

[leiningen]: https://github.com/technomancy/leiningen

## Running

```shell
$ export GITHUB_USERNAME=samrat
$ export GITHUB_TOKEN=...
$ lein run
```
